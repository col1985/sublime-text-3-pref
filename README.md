Sublime Text 3 Preferences
====

Customized personal Sublime Text 3 preferences and setting for [PreDawn](https://github.com/jamiewilson/predawn) theme for Sublime Text 3. 

####List of Plugins 

+ Emmet
+ JsFormat
+ bracketHighlighter
+ Markdown Preview
+ SublimeLinter
+ JsHint Gutter
+ Side Bar
+ Less
+ DocBlockr
